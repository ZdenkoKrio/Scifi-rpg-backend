This is the API documentation and testing interface for the Scifi RPG backend. Below you will find links to different API modules.

📖 API Modules

  • 🛠 Items API - Manage weapons, armor, and consumables.
  • 🧑‍🚀 NPC API - Interact with merchants, enemies, and quest givers.
  • 🎮 Player API - Manage player stats, inventory, and reputation.
  • 🚀 Ship API - Manage spaceships, weapons, and crew.
  • 🌌 Space API - Explore star systems, planets, and anomalies.

📡 How to Use

Each module provides a testable API overview page where you can interact with different endpoints using GET requests.

⚡ Example Request

Fetch all items:

fetch('/api/items/')
    .then(response => response.json())
    .then(data => console.log(data))
